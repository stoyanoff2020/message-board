## Getting Started

The Message Board is a web application where registered users can publish and browse messages. Each message contains:
- Title
- Description
- Publisher name (automatically set to your name)
- Contact phone number
- Publication date (automatically set)

### System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- JavaScript enabled

## User Registration

To start using the message board, you need to create an account.

### How to Register

1. Navigate to the registration page by clicking "Register" in the navigation
2. Fill out the registration form:
   - **Email**: Enter a valid email address
   - **Password**: Must be at least 8 characters long
   - **Name**: Your full name (this will appear as the publisher on your messages)
3. Click "Register" to create your account
4. You'll be automatically logged in after successful registration

### Registration Requirements

- **Email**: Must be a valid email format and not already registered
- **Password**: Minimum 8 characters for security
- **Name**: Required field that will be displayed on your published messages

## User Login

### How to Login

1. Click "Login" in the navigation
2. Enter your registered email and password
3. Click "Login" to access your account
4. You'll be redirected to the main message board

### Session Management

- Your session will remain active until you logout or it expires
- If your session expires, you'll be redirected to the login page
- Click "Logout" in the navigation to end your session

## Publishing Messages

Once logged in, you can publish messages for other users to see.

### How to Publish a Message

1. Navigate to the main message board page
2. Fill out the message form:
   - **Title**: A clear, descriptive title for your message
   - **Description**: Detailed information about your message
   - **Contact Phone**: Your phone number for interested parties to reach you
3. Click "Publish Message" to submit

### Message Requirements

- **Title**: Required, maximum 255 characters
- **Description**: Required, can be longer text
- **Contact Phone**: Required, must be a valid phone number format
- **Publisher Name**: Automatically set to your registered name
- **Publication Date**: Automatically set to current date and time

### Message Validation

The system validates your input:
- All fields must be filled out
- Phone number must be in a valid format
- If validation fails, you'll see error messages explaining what needs to be corrected

## Viewing Messages

### Message Display

All published messages are displayed on the main page, showing:
- Message title
- Full description
- Publisher's name
- Contact phone number
- Publication date

### Message Ordering

Messages are displayed with the newest messages first (most recent at the top).

### Empty State

If no messages have been published yet, you'll see a message indicating the board is empty.

## Searching Messages

You can search through all published messages to find specific information.

### How to Search

1. Use the search bar at the top of the message list
2. Type your search terms
3. Results will appear automatically as you type
4. Search terms will be highlighted in the results

### Search Functionality

- **Search Scope**: Searches both message titles and descriptions
- **Real-time**: Results update as you type
- **Highlighting**: Matching terms are highlighted in yellow
- **Case Insensitive**: Search is not case-sensitive

### Search Tips

- Use specific keywords for better results
- Try different variations of words
- Clear the search box to see all messages again

### No Results

If no messages match your search terms, you'll see a "No results found" message.

## User Account Management

### Logout

To logout of your account:
1. Click "Logout" in the navigation
2. You'll be redirected to the login page
3. Your session will be terminated

### Account Security

- Always logout when using shared computers
- Don't share your login credentials
- Contact an administrator if you suspect unauthorized access
